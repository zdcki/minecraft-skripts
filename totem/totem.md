# Custom Totems: 37 Кастомных Тотемов Возрождения

**Skript-система для Minecraft 1.21+**

Полностью кастомизируемые Тотемы Возрождения с уникальными эффектами. 37 разных тотемов, каждый со своим Custom Model Data, анимациями, звуками и способностями. Поддержка покупки за валюту, обмена 2→1 и удобной выдачи.

## Описание

Обычный тотем неуязвимости теперь превращается в 37 мощных артефактов. При воскрешении скрипт определяет Custom Model Data и выдаёт соответствующий эффект: от берсерка и вампира до остановки времени и призыва големов. Есть система покупки для донатеров и GUI-обмен.

Отлично подходит для RPG, выживания или PvP-серверов, где нужны крутые уникальные предметы.

## Возможности

- 37 уникальных тотемов (ID 1001–1037)
- Автоматическое определение тотема в главной или левой руке
- Полностью кастомные названия, лор и модельки
- Эффекты от простых (сила, скорость) до сложных (призыв големов, остановка времени, кража ХП)
- Команда `/givetotem` с автодополнением
- Покупка топовых тотемов за валюту (`/buytotem`)
- GUI-обмен «2 тотема → 1 любой обычный» (`/totemtrade`)
- Метаданные `last_attacker` для некоторых эффектов

## Как это работает

При событии `on resurrect` скрипт проверяет Custom Model Data в правой руке, потом в левой. Если ID в диапазоне 1001–1037 — срабатывает нужный эффект.  
Обмен работает через отдельное 6-строчное GUI: кладёшь два кастомных тотема слева → выбираешь любой обычный справа.  
Покупка снимает деньги через `player's balance` и вызывает `/givetotem`.

## Зависимости

- **Skript** 2.13.0+
- **SkBee** (для GUI обмена)
- Плагин экономики (Vault + любой economy)
- `simpleplayerfreeze` (для тотема Хакера)
- Плагин варпов (для тотема Побега — `warps ambar`)
- Команда `dm` (DeluxeMenus или аналог — для кнопки «Назад»)

2. Выполните:
