
# Custom Totems

## Описание

Скрипт добавляет в игру 37 кастомных тотемов возрождения с уникальными эффектами. Обычный тотем превращается в мощные артефакты с разными способностями: от усиления урона и вампиризма до остановки времени и призыва мобов.

Подходит для RPG, survival и PvP-серверов, где важны уникальные предметы и механики.

## Возможности

* 37 уникальных тотемов (ID `1001–1037`)
* автоматическое определение тотема в основной или левой руке
* кастомные названия, лор и Custom Model Data
* широкий спектр эффектов:

  * базовые (сила, скорость, защита)
  * продвинутые (кража ХП, призыв големов, остановка времени)
* команда выдачи `/givetotem` с автодополнением
* покупка топовых тотемов через `/buytotem`
* GUI-обмен `2 → 1` (`/totemtrade`)
* использование метаданных `last_attacker` для некоторых механик

## Как это работает

При событии `on resurrect`:

1. Проверяется предмет в правой руке
2. Если не найден — проверяется левая
3. Если `Custom Model Data` в диапазоне `1001–1037` — применяется соответствующий эффект

Обмен реализован через GUI:

* игрок кладёт 2 кастомных тотема в левую часть
* выбирает любой обычный тотем справа

Покупка:

* снимает деньги через `player's balance`
* вызывает команду `/givetotem`

## Зависимости

* Skript `2.13.0+`
* SkBee (для GUI)
* экономика (Vault + любой economy-плагин)
* `simpleplayerfreeze` (для тотема Хакера)
* плагин варпов (для тотема Побега — `warps ambar`)
* команда `dm` (DeluxeMenus или аналог — кнопка «Назад»)

## Установка

1. Положить файл `totems.sk` в папку:

   ```
   plugins/Skript/scripts/
   ```

2. Перезагрузить скрипт:

   ```bash
   /sk reload totems
   ```

## Использование

### Игроки

* использовать тотем с кастомной моделью — эффект сработает автоматически
* `/buytotem <тип>` — купить топовый тотем (право `dzuni.skript`)
* `/totemtrade` — открыть меню обмена

### Администраторы

* `/givetotem <тип> [игрок]` — выдать тотем

## Настройка

### `options:`

Все ID задаются в начале скрипта:

```yaml
options:
    berserk_id: 1001
    turtle_id: 1002
    phantom_id: 1003
    # ...
    overlord_id: 1037
```

* можно менять ID под свои модели
* названия и цены задаются внутри `/givetotem` и `/buytotem`

## Примечания

* тотемы работают только при стандартном механизме воскрешения Minecraft
* некоторые тотемы требуют сторонние плагины:

  * Хакер
  * Побег
  * Оверлорд
* обмен запрещает использование эпических тотемов (god, overlord и т.д.)
* `last_attacker` обновляется при любом уроне от игрока
* весь скрипт локализован на русский язык

## Расширение

Чтобы добавить новый тотем:

1. Скопировать обработку в `on resurrect`
2. Добавить выдачу в `/givetotem`
3. (опционально) добавить в `/buytotem`
