# Свиток клана

## Описание

Скрипт добавляет квестовую систему доставки: игрок берет задание у NPC `Хироши`, получает свиток и должен донести его до одного из случайных получателей. На маршруте мешают таймер, случайные события, штрафы за урон и вода, а в конце выдаётся награда и записывается статистика.

Подходит для серверов с ролевой подачей, доставками, клановой тематикой и мини-ивентами по пути.

## Возможности

* 2 режима доставки:

  * `Обычная доставка`
  * `Контрабанда`
* случайный NPC-получатель из 4 вариантов:

  * Такеши
  * Акира
  * Кенджи
  * Юми
* выдача квестового предмета `Свиток Клана` с `custom model data`
* компас-указатель на цель
* сохранение и возврат инвентаря игрока на время задания
* случайные ивенты во время доставки: от баффов до опасных ситуаций
* штраф времени за получение урона
* штраф времени за нахождение в воде
* запрет на команды во время доставки
* запрет на полёт, транспорт, дроп и перенос квестовых предметов
* штраф/провал при смерти
* система преследователя для режима `Контрабанда`
* расчёт награды по оставшемуся времени
* личная статистика доставок
* топ-5 курьеров через команду и PlaceholderAPI

## Как это работает

Игрок подходит к NPC `Хироши` и выбирает режим. После старта скрипт:

* сохраняет инвентарь, броню и оффхенд
* выдаёт свиток и компас
* ставит цель на одного из четырёх NPC
* запускает таймер

Пока задание активно, каждый тик ивент-система может подкинуть случайный эффект. Это могут быть как плюсы, так и штрафы, поэтому маршрут лучше не растягивать.

Награда считается от оставшегося времени:

* в обычном режиме — до `¥5000`
* в контрабанде — до `¥10000`

Минимальная выплата не опускается ниже `¥100`.

## Зависимости

* Skript
* экономический плагин с поддержкой `balance` / `remove from balance` и `add to balance`
* WorldGuard-совместимая поддержка регионов, потому что используется `region at player`
* NPC-аддон для Skript с событием `on npc right click` и `event-npc`
* PlaceholderAPI
* аддон с поддержкой `custom model data` для предметов

## Установка

1. Положить файл `svitok.sk` в `plugins/Skript/scripts/`.
2. Перезагрузить скрипты:

```bash
/sk reload svitok
```

## Использование

Основной вход в систему — NPC `Хироши`.

Игрок:

* нажимает ПКМ по `Хироши`
* выбирает режим в меню
* получает свиток и компас
* несёт предмет к указанному NPC
* сдаёт квест, держа свиток в инвентаре

Команды:

```bash
/zevent <id>
/deliverystats
/deliverytop
/dtop
/топдоставок
```

`/zevent` — debug-команда для запуска конкретного ивента от 1 до 110.
`/deliverystats` — личная статистика.
`/deliverytop` — топ-5 курьеров.

## Настройка

### `options:`

```skript
npc_loc_1: location(10206, 140, -122997, world "postroiki")
npc_loc_2: location(10244, 134, -123001, world "postroiki")
npc_loc_3: location(10096, 71, -122587, world "postroiki")
npc_loc_4: location(10358, 56, -122661, world "postroiki")
fail_teleport_loc: location(10276, 167, -122633, world "postroiki")

penalty_amount: 1000

scroll_item: paper named "§6§lСвиток Клана" ...
compass_item: compass named "§bУказатель пути"

reward_std: 5000
time_std: 300
chance_std: 50

reward_hard: 10000
time_hard: 180
chance_hard: 100
```

### Важные переменные

* `{delivery.active::%uuid%}` — активна ли доставка
* `{delivery.mode.%uuid%}` — режим (`normal` / `hard`)
* `{delivery.timer.%uuid%}` — оставшееся время
* `{delivery.target_name.%uuid%}` — имя цели
* `{delivery.target_loc.%uuid%}` — координаты цели
* `{delivery.scroll_item.%uuid%}` — сам свиток
* `{delivery.scroll_id.%uuid%}` — ID модели свитка
* `{delivery.stats.money::%uuid%}` — заработано денег
* `{delivery.stats.count::%uuid%}` — количество выполненных доставок
* `{delivery.stats.name::%uuid%}` — имя игрока для базы статистики

### PlaceholderAPI плейсхолдеры

* `delivery_player_money`
* `delivery_player_count`
* `delivery_player_rank`
* `delivery_top_name_1` ... `delivery_top_name_5`
* `delivery_top_money_1` ... `delivery_top_money_5`
* `delivery_top_count_1` ... `delivery_top_count_5`

## Примечания

* Во время доставки игрок не может использовать команды, открывать контейнеры, летать, садиться в транспорт и выбрасывать квестовые предметы.
* Урон режет таймер на `10 секунд`, вода — на `5 секунд`.
* При смерти доставка проваливается, вещи возвращаются, а игрок получает штраф.
* Для контрабанды спавнится преследователь-скелет с невидимостью и мечом.
* В коде жёстко завязаны имена NPC и GUI; если менять мир или координаты, нужно править `options:`.
* Логика ивентов содержит 110 вариантов, часть из них полезные, часть откровенно вредные. Это сделано намеренно.
