# 🐄 ambar.sk — Система животноводства

Скрипт для амбара/фермы: игроки разводят животных в специальном регионе, копят отчёты и сдают их за деньги. Есть прогрессия рангов, витаминный корм для мгновенного роста молодняка и зона забоя с красивыми эффектами.

---

## 🐾 Что умеет

- **Разведение с прогрессией** — каждое размноженное животное в регионе `ambar` идёт в отчёт игрока
- **3 ранга фермера**: Новичок (×150¥) → Фермер от 500 голов (×200¥) → Мастер от 2000 голов (×300¥)
- **Двойня** — у Мастеров 10% шанс родить двух детёнышей вместо одного
- **Золотое животное** — шанс 0.01% получить светящегося зверя + легендарный ключ
- **Витаминный корм** (`/buyfood`) — покупаешь за 500¥/шт., кликаешь на детёныша, он мгновенно вырастает
- **Зона забоя** — жмёшь кнопку, все животные в регионе `slaughter_zone` убиваются с партикл-эффектами, деньги начисляются автоматически (кулдаун 5 минут)
- **PlaceholderAPI** — плейсхолдеры `%breed_balance%`, `%breed_total%`, `%breed_rank%`, топ-10 через `%breedtop_name_1%` / `%breedtop_value_1%`

## 📦 Зависимости

| Плагин | Для чего |
|--------|----------|
| [Skript](https://github.com/SkriptLang/Skript) | Основа |
| WorldGuard | Регионы `ambar` и `slaughter_zone` |
| Vault + EssentialsX | Работа с балансом |
| PlaceholderAPI + skript-placeholders | Плейсхолдеры и топ |
| ItemsAdder *(опционально)* | Кастомная модель корма (custom model data) |

---

## ⚙️ Установка

1. Скопировать `ambar.sk` в `/plugins/Skript/scripts/`
2. Создать регионы `ambar` и `slaughter_zone` через WorldGuard
3. Поставить каменную кнопку на координатах из `on load:` (или поменять координаты под свой сервер)
4. `skript reload ambar`

**Что подстроить под себя:**
- Координаты кнопки в `on load:`
- Название мира (`world "postroiki"`)
- Цены и custom model data корма в `options:`
