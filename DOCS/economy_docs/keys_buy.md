# 🔑 keys_buy.sk — Система покупки ключей через чат

Продвинутый скрипт для продажи кейсов или ключей, который избавляет игрока от лишних кликов в интерфейсе. Вместо выбора фиксированного количества в меню, игрок просто вводит нужное число прямо в чат, что делает процесс покупки максимально быстрым и гибким.

---

## 🚀 Что умеет

- **Интерактивный ввод через чат (`AsyncPlayerChatEvent`)** — главная особенность скрипта. После выбора типа ключа в GUI, скрипт переходит в режим ожидания: чат для игрока временно блокируется, и он может просто написать цифру (количество). Это реализовано через глубокую интеграцию с событиями Bukkit для мгновенного отклика.
- **Умная обработка ввода** — скрипт сам распознает отмену покупки (если написать "отмена") или некорректные данные, возвращая игрока в обычный режим чата без ошибок.
- **Автоматический расчет стоимости** — система сама умножает цену на введенное количество, проверяет баланс и выдает товар одной консольной командой.
- **Очистка данных экономики** — встроенный алгоритм очистки строк (удаление запятых, символов валют и пробелов) гарантирует, что расчет баланса пройдет корректно на любом плагине экономики.
- **Визуальный фидбек** — использование Action Bar для уведомлений о покупке и приятные звуковые эффекты (Amethyst и Level Up) при успешной сделке.

## ⚙️ Установка

1. Скопировать `keys_buy.sk` в папку `/plugins/Skript/scripts/`.
2. Убедиться, что установлен аддон **skript-reflect**, так как импорт событий чата идет напрямую из ядра (org.bukkit.event).
3. Выполнить команду: `/sk reload keys_buy`.
4. Настроить команду выдачи ключей в блоке `buykeys_internal` (по умолчанию используется `ac:ac key give`).

**Что подстроить под себя:**
- Команду выдачи ключа в зависимости от твоего плагина на кейсы.
- Сообщения и префиксы в блоке «ПЕРЕХВАТ ЧАТА».
- Звуковые эффекты в конце успешной транзакции.
