# ⛏️ pickaxeshop.sk — Магазин инструментов

Полноценная шахтёрская экосистема для Minecraft-сервера: магазин кирок, скупщик ресурсов, система шахт и бустеры. Всё в одном скрипте — от покупки первой алмазки до легендарного «Сердца сервера» за 200к.

---

## 🛒 Что умеет

### Магазин кирок (`/pickaxeshop`)
- 14 кирок: от алмазки с Efficiency II до незеритки с Fortune III + Mending
- Легендарные предметы: **Разрушитель Спавнеров** (ломает спавнер с сохранением типа) и **Разрушитель Бедрока**
- При покупке «Сердца сервера» — серверный анонс всем игрокам
- Владелец вписывается в лор кирки автоматически
- **Сердце сервера** даёт шанс 10% на Haste II при добыче

### Скупщик ресурсов (`/sellgui`)
- 19 типов ресурсов: от булыжника до древних обломков
- Продажа по 1 / 16 / 32 / 64 / всё — переключается ПКМ
- Авто-продажа на каждый предмет отдельно (клавиша Q) или сразу на всё
- Кнопка «Продать весь инвентарь» — вычищает сразу всё за один клик
- Статистика заработка и текущий множитель прямо в GUI

### Система множителей и бустеров (`/ba`)
- Донат-ранги получают базовый бонус к выплатам (x1.2 / x1.5)
- Глобальный бустер: включается для всего сервера с таймером
- Личный бустер: выдаётся конкретному игроку
- Отображение оставшегося времени бустера в GUI

### Шахты (`/mineszxc`)
- 6 видов шахт: Обычная, Редкая, Строительная, Железная лихорадка, Эпическая, Алмазная
- У каждой — свой состав блоков с процентами
- При покупке дорогой шахты (80k+) — серверный анонс

## 📦 Зависимости

| Плагин | Для чего |
|--------|----------|
| [Skript](https://github.com/SkriptLang/Skript) | Основа |
| [skript-gui](https://github.com/APickledWalrus/skript-gui) | GUI инвентари |
| Vault + EssentialsX | Работа с балансом игроков |
| WorldGuard | Регионы для спавнеров (если используется) |
| AutoMine (или аналог) | Смена режима шахты через команду |

---

## ⚙️ Установка

1. Убедиться, что все зависимости установлены
2. Скопировать `pickaxeshop.sk` в папку `/plugins/Skript/scripts/`
3. Выполнить в консоли: `skript reload pickaxeshop`
4. Настроить цены под сервер — всё задаётся в блоке `on load:`

## 💬 Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/pickaxeshop` | Открыть главное меню магазина | Все |
| `/sellgui` | Открыть скупщик ресурсов | Все |
| `/mineszxc` | Меню покупки шахты | Все |
| `/ba set <время>` | Включить глобальный бустер | OP |
| `/ba personal <ник> <время>` | Выдать личный бустер | OP |
| `/ba info` | Показать активные бустеры | OP |
| `/ba resetall` | Сбросить все бустеры | OP |
