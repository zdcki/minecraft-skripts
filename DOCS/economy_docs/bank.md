# 🏦 bank.sk — Система банковских кнопок

Скрипт для создания интерактивных зон в банке: последовательная активация механизмов через нажатие кнопок. Игроки могут по очереди активировать редстоун-сигналы по заданным координатам, что идеально подходит для открытия хранилищ или запуска анимаций.

---

## 🚀 Что умеет

- **Последовательная активация** — каждое нажатие переключает активацию на следующую точку из списка (цикличный режим).
- **Редстоун-сигнал на время** — при нажатии в целевой локации на 1 секунду появляется редстоун-блок, который затем автоматически исчезает.
- **Поддержка нескольких кнопок** — можно настроить несколько физических кнопок (например, вишневые кнопки), работающих с общим списком активаций.
- **Защита от спама** — встроенный кулдаун в 1 секунду между нажатиями для предотвращения лагов механизмов.
- **Ограничение доступа** — работа с кнопками доступна только игрокам с определенным правом (permission).
- **Инфо-команды** — перенаправление игроков на нужный варп при попытке использовать старые команды банка.

## ⚙️ Установка

1. Скопировать `bank.sk` в папку `/plugins/Skript/scripts/`.
2. В секции `on load:` прописать координаты ваших кнопок (`{buttonLoc-11}`, `{buttonLoc-12}`).
3. В массив `{activationLocs::*}` добавить координаты точек, где должен появляться редстоун-блок.
4. Выполнить команду: `/sk reload bank`.

**Что подстроить под себя:**
- Координаты кнопок и точек активации в `on load:`.
- Мир, в котором находятся механизмы (по умолчанию `"world"`).
- Время задержки появления редстоун-блока (`wait 1 second`).

## 💬 Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/bank` | Подсказка о местонахождении банка | Все |
| `/bankplus:bank` | Перенаправление на `/warp bank` | Все |

**Права:**
- `bank.button` — позволяет нажимать на кнопки активации механизмов.
