# 💸 atm.sk — Удобный банкомат для твоего сервера

[cite_start]Скрипт добавляет на сервер функциональный банкомат с симпатичным GUI-меню, где игроки могут перевести свои цифровые накопления в хрустящие купюры[cite: 1, 7]. [cite_start]Главная фишка — возможность снимать деньги как поштучно кликом в меню, так и целыми пачками через быструю команду[cite: 1, 10, 18].

## 🚀 Что внутри

* [cite_start]**Интуитивное GUI-меню** — 3 ряда слотов с кастомными иконками купюр и золотым слитком для массового вывода[cite: 7, 8, 10].
* [cite_start]**Снятие пачками** — команда `/take` позволяет забрать сразу гору денег (например, 64 штуки по 5000¥), чтобы не кликать в меню до посинения[cite: 1, 11].
* [cite_start]**Интерактивная справка** — внутри меню есть книжка с подсказками, а при клике на "Снять пачку" в чат выводится кликабельная инструкция, которая сама подставит начало команды в строку ввода[cite: 9, 12, 15].
* [cite_start]**Поддержка Custom Model Data** — для купюр используются кастомные текстуры (ID 10201), так что в инвентаре они будут выглядеть как настоящие деньги, а не просто бумага[cite: 7, 8].
* [cite_start]**Защита от ошибок** — скрипт проверяет баланс игрока, правильность номинала и не дает забрать декоративное стекло из меню[cite: 4, 6, 11].

## 📦 Зависимости

| Плагин | Зачем нужен |
| :--- | :--- |
| **Skript** | Основа, на которой написан код. |
| **Vault + EssentialsX** | [cite_start]Чтобы скрипт видел баланс игрока и мог списывать деньги[cite: 6]. |
| **pmoney** (или аналог) | [cite_start]Команда, через которую выдаются сами предметы валюты[cite: 6]. |
| **ItemsAdder / Oraxen** | [cite_start]Чтобы бумажки превратились в красивые купюры через Custom Model Data[cite: 7, 8]. |

## ⚙️ Установка

1. Закинь файл `atm.sk` в папку `/plugins/Skript/scripts/`.
2. [cite_start]Убедись, что у тебя настроена выдача купюр через команду `/pmoney take [номинал]`[cite: 6].
3. Пропиши в консоли или в игре `/sk reload atm`.
4. Готово! [cite_start]Чтобы открыть меню, введи `/atm` или привяжи эту команду к любому NPC или блоку[cite: 1].

## 💬 Команды

* [cite_start]`/atm` — открывает главное меню банкомата[cite: 1].
* [cite_start]`/take <номинал> <количество>` — быстрое снятие нужной суммы (доступные номиналы: 100, 500, 1000, 5000, 10000)[cite: 1, 5].
