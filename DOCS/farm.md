# 🥔 farm.sk — Продвинутая система фермерства

Скрипт превращает обычный сбор урожая в полноценную игровую механику с динамической экономикой, прокачкой хранилища и системой рангов. Игроки собирают картофель в специальной зоне, следят за курсом цен и улучшают своё оборудование для максимальной прибыли.

---

## 🚀 Что умеет

- **Динамический рынок** — каждые 10 минут курс цен на картофель меняется (от 0.8x до 1.5x), о чём сервер уведомляет всех игроков в чате.
- **Прокачка корзины** — у каждого игрока есть лимит хранилища. Его можно улучшать за игровую валюту, увеличивая вместимость для собранного урожая.
- **Система рангов** — за общее количество собранного картофеля игроки получают титулы: от «Новичка» до «Мастера фермерства».
- **Умная мотыга** — специальный инструмент, который автоматически отправляет урожай в виртуальную корзину при вскапывании грядок в регионе `potat`.
- **GUI-меню** — удобный интерфейс для бесплатного получения фермерской мотыги через команду `/hoemenu`.
- **Интеграция с PlaceholderAPI** — вывод статистики игрока (ранг, баланс корзины, общий сбор) в скорборд или таб.

## 📦 Зависимости

| Плагин | Для чего |
|--------|----------|
| [Skript](https://github.com/SkriptLang/Skript) | Основа логики |
| WorldGuard | Регион `potat` для работы фермы |
| Vault + EssentialsX | Работа с экономикой и выплатами |
| PlaceholderAPI | Отображение статистики через плейсхолдеры |

---

## ⚙️ Установка

1. Скопировать `farm.sk` в папку `/plugins/Skript/scripts/`.
2. Создать регион с названием `potat` через WorldGuard в зоне фермы.
3. Выполнить команду: `/sk reload farm`.
4. Настроить базовые награды и стоимость улучшения в секции `options:` внутри файла.

**Что подстроить под себя:**
- `region_name`: название региона фермы.
- `base_reward`: базовая выплата за картофель.
- `exchange_cost`: стоимость обмена/улучшения.
