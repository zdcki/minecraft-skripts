
# 📦 Skript

**Коллекция готовых механик для Minecraft 1.21+**

Все скрипты написаны на **Skript 2.13.0+** с использованием **SkBee**.
Каждый проект — полностью самостоятельный, протестирован и готов к установке.

🌐 **Сайт сервера / демо:** [Poxland](https://zdcki.gitbook.io/poxland)

---

## 📜 Список скриптов

| Скрипт             | Описание                             | Основные возможности                                   | Файл             |
| ------------------ | ------------------------------------ | ------------------------------------------------------ | ---------------- |
| **TD Defence**     | Tower Defense режим в отдельном мире | 9 мобов, собака-компаньон, настройка волн, награды     | `def.sk`  |
| **OITC**           | One In The Chamber (PvP режим)       | 4 режима оружия, 3 брони, эффекты убийств, бустеры     | `oitc.sk`        |
| **Pickaxe Market** | Магазин кирок + скупщик              | GUI-магазин, авто-продажа, разрушители блоков          | `pickaxeshop.sk` |
| **RegionHack**     | Взлом и защита регионов              | ключи, уровни защиты, ловушки, глушилки                | `region.sk`  |
| **InventoryRunes** | Система пассивных рун                | 50+ рун, лимиты, кастом механики (магнит, дабл прыжок) | `runes.sk`       |
| **Свиток Клана**   | Квест доставки                       | NPC, таймер, события, штрафы, статистика + PAPI        | `svitok.sk`      |

## ⚙️ Общие зависимости

* **Skript 2.13.0+**
* **SkBee** (GUI, NBT, атрибуты)
* **Vault** + любой economy-плагин

Дополнительно:

* **WorldGuard + WorldEdit** — для `RegionHack`
* **ItemsAdder** — для некоторых предметов (`/iagive`)

## 📥 Установка

1. Скопируйте нужный `.sk` файл в:

   ```
   plugins/Skript/scripts/
   ```

2. Перезагрузите скрипт:

   ```bash
   /sk reload <имя_файла>
   ```

3. Настройте:

   * координаты
   * миры
   * регионы

(всё подробно указано в README каждого скрипта)

## 🎮 Использование

Каждый скрипт имеет собственный `README.md` с подробной документацией.

Примеры команд:

* `/defence` — войти в TD арену
* `/oitc` — открыть меню PvP режима
* `/pickaxeshop1` — магазин кирок
* `/sellgui` — скупщик ресурсов
* `/totemtrade` — обмен тотемов
* `/runeshop` — магазин рун
* ПКМ по NPC **Хироши** — начать квест со свитком

## 🧩 Особенности

* все GUI и сообщения на русском языке
