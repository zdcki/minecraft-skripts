
# RegionHack

## Описание

Скрипт добавляет систему взлома и защиты регионов на базе WorldGuard. Игроки могут взламывать чужие приваты с помощью специальных ключей, а владельцы — усиливать защиту, ставить ловушки и использовать защитные устройства.

Подходит для анархии, faction и RPG-серверов с базами и PvP-механиками.

## Возможности

* 5 уровней защиты региона (`1–5`) + абсолютная защита (`6`)
* 5 уровней хакерских ключей
* модули защиты (повышают уровень региона)
* анти-хакерская ловушка:

  * уничтожает ключ
  * вызывает удар молнии
* глушилка связи:

  * блокирует `/home`, `/spawn`, `/tpa` и связанные команды на 20 секунд
* сканер ловушек:

  * кулдаун 60 секунд
  * шанс обнаружения 50%
* декодер-имитатор (ложный взлом без траты ключа)
* автоматическое управление доступом к региону при взломе
* маяк и глобальное оповещение при взломе высоких уровней
* команда-гайд `/guidehack` (5 страниц)

## Как это работает

Скрипт использует регионы через `regions at location`.

Механика взлома:

1. Игрок нажимает ПКМ с ключом
2. Проверяется уровень ключа и уровень защиты региона
3. Если уровень достаточный — запускается таймер (`60 секунд`)
4. Игрок должен оставаться внутри региона
5. При успехе:

   * уровень защиты уменьшается на 1
   * игрок добавляется в регион (`member`)

Если игрок выходит из региона или умирает — взлом прерывается.

Дополнительно:

* ловушки, глушилка и декодер работают через отдельные предметы
* все действия происходят в реальном времени с эффектами и уведомлениями

## Зависимости

* Skript `2.13.0+`
* WorldGuard
* WorldEdit
* SkBee (рекомендуется для стабильной работы)

## Установка

1. Положить файл `regionhack.sk` в папку:

   ```
   plugins/Skript/scripts/
   ```

2. Перезагрузить скрипт:

   ```bash
   /sk reload regionhack
   ```

3. Убедиться, что:

   * WorldGuard установлен
   * регионы уже созданы

## Использование

### Хакеры

* взять «Хакерский Ключ» нужного уровня
* ПКМ по региону → начать взлом
* `/rglevel` — проверить уровень защиты региона

### Владельцы региона

* ПКМ «Модуль Защиты» — повысить уровень региона

* ПКМ «Анти-хакерская ловушка» — установить ловушку

* ПКМ «Глушилка Связи» — временно отключить телепорты

* `/rgprotect` — включить / выключить абсолютную защиту (админ)

### Администраторы

* `/hackitems` — получить все предметы системы
* `/hacktoggle` — включить / выключить систему взлома

## Настройка

### `options:`

```yaml id="p9x2k1"
options:
    prefix: &8[&6&lHACK&8] &f
    hack_time: 60
    jam_time: 20
```

* `hack_time` — длительность взлома (сек.)
* `jam_time` — длительность глушилки (сек.)

Уровни защиты хранятся в:

```
{region.protect::%regionname%}
```

## Примечания

* система работает только внутри регионов WorldGuard
* абсолютная защита (`уровень 6`) делает регион невзламываемым
* при выходе из региона или смерти взлом прерывается
* глушилка блокирует команды:

  * `/home`
  * `/spawn`
  * `/tpa`
  * `/tpaccept`
  * `/tp`
* декодер использует тот же флаг, что и обычный взлом:

  ```
  {hacker::%uuid%}
  ```
* для отладки используется функция `debug()` (отправляет сообщения игроку)

## Расширение

Чтобы добавить новые механики:

1. Скопировать шаблон предмета из `/hackitems`
2. Добавить обработку в `on right click`
3. (при необходимости) расширить систему уровней защиты
