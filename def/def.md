
# TD Defence

## Описание

Скрипт добавляет полноценный режим Tower Defense в отдельном мире `voidworld`. Игроки защищают базу от волн мобов, которые движутся по фиксированному маршруту.

Дополнительно реализована система прокачиваемого питомца (собаки), настройка скорости волн через GUI и награды за убийства.

Подходит для мини-игр, survival-серверов и отдельных игровых режимов.

## Возможности

* фиксированный путь из 6 точек
* 9 типов мобов с разными характеристиками:

  * здоровье
  * скорость
  * штраф за прорыв
* автоматический спавн волн (рандом + таймер)
* настройка скорости волн через GUI (за валюту)
* боевой компаньон (собака):

  * покупка
  * прокачка урона
  * прокачка скорости
* награды за убийства:

  * деньги
  * золотые самородки
* автосохранение и восстановление инвентаря
* GUI-меню управления
* автоматический выход при смерти
* админ-команды для полного контроля режима

## Как это работает

Механика волн:

* мобы спавнятся в первой точке
* двигаются по маршруту из 6 точек
* используется:

  * `NoAI`
  * `NoGravity`
  * ручной телепорт каждый тик

Если моб доходит до конца:

* все игроки в радиусе 30 блоков получают штраф (деньги снимаются)

Собака:

* автоматически появляется при входе на арену
* атакует только TD-мобов
* урон и скорость можно улучшать за деньги

Скорость волн:

* регулируется через GUI
* диапазон: `1–10 секунд`

## Зависимости

* Skript `2.13.0+`
* SkBee (для GUI)
* экономика (Vault + любой economy-плагин)
* ItemsAdder (используется `/iagive`)
* мир `voidworld`

## Установка

1. Положить файл `td-defence.sk` в папку:

   ```
   plugins/Skript/scripts/
   ```

2. Перезагрузить скрипт:

   ```bash
   /sk reload td-defence
   ```

3. При необходимости настроить путь мобов в `on load`

## Использование

### Игроки

* `/defence` — войти на арену
* убивать мобов → получать деньги и награды
* ПКМ по предмету «Настройка TD» — открыть меню скорости
* `/td-dog` — призвать или убрать собаку
* `/td-dog-up` — открыть меню прокачки
* слот 8 (барьер) — выход с арены

### Администраторы

* `/td-start <число>` — запустить волны
* `/td-stop` — остановить
* `/td-speed <1-10>` — изменить скорость
* `/td-menu` — открыть GUI (даже вне арены)
* `/td-dog-reset <игрок>` — сброс прокачки

## Настройка

### Основные параметры

```yaml id="td82kx"
on load:
    # точки пути (пример)
    add location(358.5, 31, -102.5, world "voidworld") to {td.path::*}
    # ...

    set {td.spawn_interval} to 5
```

* `{td.path::*}` — точки маршрута
* `{td.spawn_interval}` — интервал спавна (в секундах)

Дополнительно:

* награды задаются в `on death`
* цены на прокачку — в GUI-функциях

## Примечания

* работает только в мире `voidworld`
* мобы двигаются через:

  * `NoAI`
  * `NoGravity`
  * телепорт по точкам
* собака атакует только TD-мобов (через метаданные)
* при выходе игрока:

  * он удаляется с арены
  * инвентарь возвращается
* если игроков нет:

  * спавнер отключается через 10 секунд
* все GUI и сообщения на русском языке

## Расширение

Чтобы добавить новых мобов:

1. Добавить моба в функцию `spawnTDMob`
2. Прописать награды в `on death`
3. При необходимости задать уникальные параметры
